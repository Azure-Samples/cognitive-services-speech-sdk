Prerequisites

Before you begin building and using V2TIC, you'll need these tools and libraries for building and deploying:

  • Docker. From instructions for preparing Docker for V2TIC, go to Dockerfile in this project.
  • minikube
    Note: This documentation uses minikube to set up and run a local Kubernetes environment. However, you can use other services instead.
  • kubectl
    • to verify and get information about builds from your CLI (you can also use Kubernetes Dashboard)
  • An interface for configuring and running commands:
    • For Unbuntu users, we recommend:
      • Make
      • Shell
      • envsubst
    • For Windows users, we recommend installing Linux through WSL, then building through that subsystem.
  • The Python libraries listed in requirements.txt.

You also need to have your Azure AI for Text to Speech resources set up, including:

  • An Azure AI subscription for Speech.
  • An Azure AI endpoint for Speech to Text REST API for short audio.
  • The URL and key for your endpoint.

To find out more about setting up and configuring Azure AI Speech services, go to the documentation.
