Download example data used in the sample project from
https://aka.ms/embedded-speech-samples-data (zip file)

Extract the following files from the package and place them in the same folder
where this readme.txt is located:

keyword_computer.table - keyword recognition model (keyword is "Computer")
performance_test.wav   - performance test input file, WAV file format
speech_test.raw        - speech recognition test input file, raw audio format
speech_test.wav        - speech recognition test input file, WAV file format
